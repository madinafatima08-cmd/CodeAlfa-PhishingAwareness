What is Phishing?
Phishing is a cyber attack where attackers try to steal your personal information like passwords, bank details by pretending to be trusted companies or people.
Common Types of Phishing
Email Phishing: Fake emails that look real
SMS Phishing: Fake messages with suspicious links  
Website Phishing: Fake websites that copy real ones
How to Identify Phishing
Check sender email carefully
Look for spelling mistakes
Don’t click suspicious links
Hover over links to see real URL
Never share passwords via email/message
What to Do If You Get a Phishing Message
Don’t click the link
Delete the message
Report it if possible
